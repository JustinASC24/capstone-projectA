Events

Git Hub Repository
Website Link

In this project, we got introduced to DOM and how we could add behavior to the user's actions. Here, we have a color selector program comprised of many buttons that output exactly as labeled. The hide button hides everything, the color buttons output the color itself and its name, and the randomizer randomizes that process.

AI-tivity: Independent Exploration of ChatGPT

Git Hub Repository
Website Link

In this project, we learned about the functionality of AI in the world of computer science and tech, having the ability to help with code and thoroughly explain intricate concepts or in this case, code. Using AI, we were able to learn a new JS/DOM property and creates a counter that tracks and outputs how long a user spends on a website.
